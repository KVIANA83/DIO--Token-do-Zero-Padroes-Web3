# DIO - Bootcamp Blockchain Developer with Solidity 

### Desafio de Projeto
Criando o Seu Primeiro Token do Zero nos Padrões Web3

### Desafio: Criação de Token ERC-20

Este projeto guiará você na criação de um Token ERC-20 a partir do zero, seguindo as instruções fornecidas pelo Instrutor Ricardo Zago. O processo envolve os seguintes passos fundamentais:

1. **Configuração da Wallet Metamask:**
   - Configure uma nova Wallet utilizando o Metamask.
   - Conecte-se ao ambiente de desenvolvimento Ethereum configurando o RPC (Remote Procedure Call) info necessário.

2. **Desenvolvimento do Token:**
   - Utilize a IDE REMIX para realizar as alterações necessárias no código base do Token ERC-20.
   - Personalize o contrato com informações específicas do seu Token, como nome, símbolo, e número inicial de tokens.

3. **Deploy do Token:**
   - Compile o contrato na REMIX para verificar a correta sintaxe e funcionalidade.
   - Utilize o Metamask para fazer o deploy do seu Token na blockchain de teste configurada.
